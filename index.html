Officebook

A full-stack clone of Facebook, with a theme inspired by NBC's 'The Office', built primarily using React-Redux, Ruby, Ruby on Rails, PostgreSQL and AWS. Users are able to create an account and search for other users to friend request. Full CRUD (create, read, update, destroy) functionality is available for post, comments, and likes. Users can even upload their own photos!

TuneUp

A full-stack web application geared towards connecting musicians with one another by way of hosting jam sessions, otherwise referred to as TuneUps. This application was built in collaboration with other engineers using the MERN (MongoDB, Express.js, React-Redux, Node.js) stack. It leverages Googles API, allowing users to search locally for TuneUps to join or create their own!

Binary-Search-Tree-Visualizer

An educational platform which serves to elucidate the nuances of complex data structures, offering users a dynamic and visually engaging environment to comprehend the operations and attributes inherent to Binary Search Trees. This single-page web application was developed using vanilla JavaScript and HTML canvases to furnish a visually instructive sandbox environment.
